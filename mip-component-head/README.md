# mip-component-head

mip-component-head 用于网页头部导航栏切换

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-component-head/mip-component-head.js

## 示例

### 基本用法
```html
<mip-component-head>
   <ul>
      <li><a  href=''/></li>
	  <li><a  href=''/></li>
	  <li><a  href=''/></li>
   </ul>
   
    <ul class='head-nav-2'>
      <li><a  href=''/></li>
	  <li><a  href=''/></li>
      <li><a  href=''/></li>
   </ul>
</mip-component-head>
```

## 属性

### {属性名}

说明：两个ul的切换
必选项：否
类型：无
取值范围：无
单位：无
默认值：无

## 注意事项

 直接可用 无需注意
