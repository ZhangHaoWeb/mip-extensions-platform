标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-hs-toggleclass/mip-hs-toggleclass.js

```html

</mip-hs-toggleclass>
```
