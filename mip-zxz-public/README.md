# mip-zxz-public

直接引用即可，而页面要写有点击事件的id seeMore 和 目标id IHaveMore

标题|内容
----|----
类型|通用
支持布局|responsive
所需脚本|https://c.mipcdn.com/static/v1/mip-zxz-public/mip-zxz-public.js

## 示例

### 基本用法
```html
<mip-zxz-public>
    <script src="https://c.mipcdn.com/static/v1/mip-zxz-public/mip-zxz-public.js"></script>
</mip-zxz-public>
```
