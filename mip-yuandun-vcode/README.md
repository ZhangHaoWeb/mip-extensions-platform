# mip-yuandun-vcode

mip-yuandun-vcode 组件说明  缘盾验证码组件，操作缘盾数据时获取验证码

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-yuandun-vcode/mip-yuandun-vcode.js

## 示例

### 基本用法
```html
<mip-yuandun-vcode vcodeurl="url" class="mip-element mip-layout-container">
    <button id="vcodeBtn">按钮</button>
</mip-yuandun-vcode>
```

### {url}
说明 {数据请求地址}  必选项{是}  类型{字符串}
