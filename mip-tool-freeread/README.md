# mip-tool-freeread

mip-tool-freeread 阅读器功能，包含呼出菜单，改变主题等。

标题|内容
----|----
类型|定制
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-tool-freeread/mip-tool-freeread.js
