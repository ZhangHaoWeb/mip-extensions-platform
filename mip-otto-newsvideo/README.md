# mip-otto-newsvideo

mip-otto-newsvideo 网校新闻页播放视频

| 标题     | 内容                                                                    |
| -------- | ----------------------------------------------------------------------- |
| 类型     | 定制                                                                    |
| 支持布局 | responsive,fixed-height,fill,container,fixed                            |
| 所需脚本 | https://c.mipcdn.com/static/v1/mip-otto-newsvideo/mip-otto-newsvideo.js |

## 示例

### 基本用法

```html
<mip-otto-newsvideo>
    <script type="application/json">
        {
            "sign":"jsz",
        }
    </script>
</mip-otto-newsvideo>
```

## 属性

### script

说明：添加配置
必选项：是
类型：字符串

## 注意事项

1. 网校定制。
2. sign值模板传入
3. 兼容mip统一走 百度视频
